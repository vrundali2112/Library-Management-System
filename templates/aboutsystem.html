     Library System Management is CRUD operation platform. The given platform is for admin use only.
     Below given are some guidelines as to how to use the system.
       - Book List: It consists of List of books along with operation buttons.Create Button is to create new book. Import From Frappe is to import books from api. Also update         and delete button button to update stock and delete if book is no longer required.

       - Member List: consists of list of members where there are buttons to add, update, delete and search the member data is provided.

       - Transactions consists of list of all the books issued to the member along with their username and date of issue , expected date of return , date of return and fine.

       - Search Books provides feature of searching book by author as well as book title any one of them can be used simultaneously according to the user needs.

     How to use the System?
       - Step 1 : Whenever a member wants to issue a book Firstly search for a particular member in the member List also search for a required Book.
            If member and Book exist -> First Search for member's username in the transaction section see if the member fine is not more than 500 .Issue the book from
            issue option on the home page.
            After Issuing the book Decrease the book stock number by 1 in the Book List.
            Else if member or book doesn't exist -> Create new record and then issue the book.
       - Step 2 : Transaction is important section check for the usernames whose date of book return is more than the expected date fine should be increased by 5 rupees
             everyday. If member pays the amount add the data in member List's total pay section of particular member name.
       - Step 3 : If member wants to return the book . Add the data in return section from home page and increase the returned book stock by 1 .
